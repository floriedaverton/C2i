Aspergillus



I - DESCRIPTION
Hôtes :
Les principaux concernés sont les humains notamment immunodéprimés, les vaches, les dauphins, les oiseaux, les chevaux

Organes cibles :
Les poumons, la peau et le cerveau sont principalement touchés.

Types de symptômes :
Tout d'abord, il existe différentes Aspergilloses : Aspergillose immunp-allergique, Aspergillome, Aspergillose invasive. Les symptômes dépendent donc du type d’infection : dyspnée, toux, hémoptysie, fièvre, douleur thoracique, plaque érythémateuse, méningites.

Maladie vectorielle :
Non, il n'y a donc aucun vecteur.


II - DISTRIBUTION SPATIALE

Epidémiologie :
La transmision se fait par inhalation et le reservoir est le sol.
La source de contamination se fait par les poussières contaminées, les céréales, les légumes, les fruits moisis ou les matières organiques en décompositions.

L'Aspergillus a une répartition mondiale.
Elle touche les personnes qui ont un déficit imminutaire tel que le SIDA, les leucémies ou les greffes d‘organe.
